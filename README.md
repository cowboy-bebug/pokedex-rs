# Quickstart

- Download the binary from [release](https://github.com/cowboy-bebug/pokedex-rs/releases/download/0.1.0/pokedex).

- Or build from source and run:
  ```bash
  % git clone https://github.com/cowboy-bebug/pokedex-rs.git
  % cd pokedex-rs
  % cargo run --release
  ```
